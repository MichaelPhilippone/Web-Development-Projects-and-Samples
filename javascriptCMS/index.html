J-CMS

What is it?

J-CMS...
is a Content Management System using Javascript and DHTML!
While this may seem light-duty and a bit under-prepared, sometimes all you need is a quick little fix to keep your web pages up-to-date without having to hack away at HTML, CSS and javascript for hours on end.

J-CMS...
uses a series of templated text files, accessed with ajax to populate your pre-defined HTML elements at page-load.
It's dynamic, easy to use and (I think) fairly intuitive.

Just set up your template and include a line of Javascript code at the end of your page and you are rolling!
